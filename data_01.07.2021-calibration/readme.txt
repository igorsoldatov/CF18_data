Сбор данных для калибровка инерциальных датчиков в условиях различных значений перегрузки (в диапазоне от 1.2-3.0 g, с шагом 0.2).

raw_calib_data - каталог с данными с датчиков
raw_calib_data/gyro_accel-0-1-.csv - файл с данными датчиков, здесь 0 - номер датчика, 1 - номер пакета с данными, каждый файл содержит 10000 записей
архив_с_ЗТП - каталог с данными с системы записи технологический параметров ЦФ-18
process_calibration_data.py - скрипт для графиков
